# System Monitor

Утилита мониторинга системных ресурсов в экосистеме Linux. Приложение в реальном времени отображает загрузку процессора, использование памяти, состояние дисков, сетевую активность и список запущенных процессов.

## Возможности

- **Процессор** — общая загрузка и по каждому ядру, графики за последние 60 секунд
- **Память** — использование RAM и swap, прогресс-бары, детализация
- **Сеть** — скорость приёма и отправки данных, общий трафик
- **Диски** — занятое место по разделам, статистика ввода/вывода
- **Процессы** — топ-10 по загрузке CPU и по использованию памяти
- Автообновление каждую секунду
- Тёмная тема интерфейса

## Требования

- Linux (доступ к `/proc`)
- Python 3.10+
- Tkinter (обычно предустановлен, иначе: `sudo apt install python3-tk`)

## Установка и запуск

### Способ 1. Из репозитория

```bash
git clone https://github.com/f1ndycs/system-monitor.git
cd system-monitor
pip install -r requirements.txt
python main.py
```

### Способ 2. Через установочный пакет

```bash
pip install dist/system_monitor-1.0.0.tar.gz
system-monitor
```

### Способ 3. Скрипт запуска

```bash
chmod +x run.sh
./run.sh
```

## Структура проекта

```
system-monitor/
├── collector/          # Модули сбора метрик (CPU, память, диски, сеть, процессы)
├── ui/                 # Графический интерфейс (CustomTkinter + Matplotlib)
├── logger/             # Логирование работы приложения
├── tests/              # Тесты для модулей collector
├── main.py             # Точка входа
├── setup.py            # Установочный пакет
├── requirements.txt    # Зависимости
├── Dockerfile          # Сборка Docker-образа
└── run.sh              # Скрипт быстрого запуска
```

## Docker (демонстрационный режим)
